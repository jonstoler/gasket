GASKET

current version: 0 (proof of concept/wip)

gasket provides lightweight lua desktop widgets

gasket runs lua scripts with functions for drawing on the root X11 window.

supported features:
* draw rectangles

eventual features:
* set widget position
* user-friendly, easy-to-use API
* draw lines
* draw text
* draw images
* read zipped archives (for easy widget packaging)
* draw circles
* draw polygons
* draw svg

gasket is meant to (eventually) be a lighter, easier to use version of conky. it is inspired by zooper for android.
